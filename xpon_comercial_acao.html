Configurar Envio de E-mail
Configure o EmailJS para enviar diagnósticos automaticamente. Crie uma conta gratuita em emailjs.com (200 e-mails/mês grátis).
Passo a passo
1. Crie conta em emailjs.com → 2. Add Service (Gmail ou outro) → copie o Service ID
3. Create Template → defina o template → copie o Template ID
4. Account → API Keys → copie a Public Key
